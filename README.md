# 大学生个人助手

大学生个人助手是一个前后端分离 Web 应用，用于管理课程、日程、学习任务、健康记录、消费预算、目标计划和习惯打卡。

## 功能模块

- 登录注册与个人中心
- 首页数据看板
- 课程与日程管理
- 学习任务分析
- 生活健康记录
- 消费与预算管理
- 个人目标与习惯打卡

## 技术栈

- 前端：Vue 3、Vite、Vue Router、Pinia、Element Plus、ECharts、Axios
- 后端：Spring Boot、Spring Security、JWT、MyBatis Plus
- 数据库：MySQL

## 启动概览

1. 使用 `backend/src/main/resources/db/schema.sql` 初始化 MySQL 数据库。
2. 在 `backend/src/main/resources/application.yml` 中配置数据库账号密码。
3. 进入 `backend` 目录运行 Spring Boot 后端。
4. 进入 `frontend` 目录安装依赖并启动 Vite 前端。

详细步骤会在项目实现完成后补充到本文档。
