# 用户使用说明

## 1. 注册和登录

打开 `http://localhost:5173` 后，进入登录页。首次使用时点击“立即注册”，填写用户名、昵称、邮箱和密码。注册成功后返回登录页，使用用户名和密码登录系统。

## 2. 首页看板

首页展示当前用户的今日课程、待办任务、健康概览、预算状态、目标进度和习惯打卡统计。系统会根据任务截止时间、预算使用率、目标进度等信息展示轻量状态提示。

## 3. 课程与日程

在“课程日程”页面可以维护三类数据：

- 课程表：新增和删除课程，记录课程名称、教师、教室、星期和节次。
- 任务 DDL：新增作业或复习任务，维护优先级、截止时间和完成状态。
- 普通日程：新增日常安排，记录地点、开始时间和结束时间。

## 4. 学习分析

在“学习分析”页面可以新增学习记录，系统会统计学习时长趋势、任务完成率、课程学习占比和高优先级任务。

## 5. 生活健康

在“健康记录”页面可以记录当天的天气、睡眠小时数、饮水量、运动分钟数、情绪评分和压力评分。同一天重复保存时，系统会自动更新原记录。

## 6. 消费预算

在“消费预算”页面可以记录收入和支出，查看支出分类统计。也可以设置月度分类预算，当支出达到阈值或超过预算时，系统会展示预警状态。同一月份同一分类重复保存预算时，系统会更新原预算。

## 7. 目标与习惯

在“目标习惯”页面可以创建阶段性目标，查看目标完成进度；也可以创建每日习惯并进行打卡，系统会统计今日打卡次数和最长连续打卡天数。

## 8. 个人中心

在右上角点击用户按钮进入个人中心，可以修改昵称、邮箱、头像地址、性别和登录密码。
